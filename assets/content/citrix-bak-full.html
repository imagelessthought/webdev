Tutorial: Using Citrix

Remote connections to Windows computers, or streamed applications, on the UNT campus are provided by Citrix Workspace. Citrix Workspace is a universal software client that provides secure, high-performance delivery of virtual desktops and applications. Clients are available for all major operating systems and accessible using any modern web browser that supports HTML5.

Please contact the UNT IT Help Desk if you have any questions, feedback, or encounter any issues with the service.

A virtual application and virtual desktop have a few differences. When you click a Virtual Application...you are only shown the application. When you use a virtual desktop, you are presented with UNT-owned software's fully functional Windows computer.

With your desktop in window mode, drag the window to span the split between the monitors you want it to use. Once it is visible on both screens, when you press the button for full screen in the toolbar, it will use all of the monitors that it is on. You may have to stretch it if you wish to use more than 2 monitors. All screens must have the exact resolution and orientation (portrait or landscape, not a combination).

There are a couple of causes for this behavior. If you have multiple monitors set at different resolutions or orientations, that can cause problems. If your monitor is set at a scaling percentage other than 100%, that can cause problems too. Opening your display settings and selecting the scale to 100% should correct that.

This document will outline how to get files to a Citrix virtual desktop from a local desktop/laptop while using Citrix Workspace app for HTML5. This article does not apply to the Citrix Workspace app for Windows/Mac users. We recommend using Google Chrome as the browser while using Citrix Workspace App for HTML5 and this document.

This will guide you on how to take a file that is currently on your Citrix Desktop and save it to your local computer.

1. From within the Citrix Desktop, click the half-circle button at the top of the window (with 3 horizontal lines) to expose the toolbar.

2. Once the Citrix toolbar is expanded, click the download button.

3. Browse the file on the Citrix desktop that you want to download to your local desktop.

4. Click the file you want to download to your local desktop and click "open".

5. Now, a window will appear on your local desktop, asking where you want to save the file. Browse to any location on your local computer where you wish to save the file and hit save. Think of this as downloading a file from a website and saving it to your local machine.

6. Once the file has been downloaded, you now have it on your local machine.

This will guide you on how to take a file currently on your local computer and save it to a Citrix Desktop.

1. From within the Citrix Desktop, click the half-circle button at the top of the window (with 3 horizontal lines) to expose the toolbar.

2. Once the Citrix toolbar is expanded, click the upload button.

3. An open file dialog will appear on your local computer. Browse to a file you wish to upload to the Citrix desktop, then click "Open."

4. From within the Citrix Desktop, choose the location you wish to save the new file. Then click "OK."

5. After a few moments, the file will appear at the location you chose on the Citrix desktop.

This document outlines installing the Citrix Workspace App for a Windows Computer. The Citrix Workspace app runs Virtual Applications & Desktops and must be installed on your computer for them to work.

1. Open your internet browser of choice (we suggest Google Chrome).

2. Navigate to https://www.citrix.com/downloads/workspace-app/windows/workspace-app-for-windows-latest.html.

3. Click the button labeled "Download Citrix Workspace app for Windows."

4. Save to your downloads folder.

5. Open your downloads folder.

6. Double click "CitrixWorkspaceApp" icon to run the installer.

7. Click "Yes" to the pop-up asking if you would like to allow the app to make changes to your computer (not everyone will get this pop-up).

8. In the "Welcome to Citrix Workspace" window, click the "Start" button.

9. Check the "I Accept the License Agreement" box on the License Agreement window and click the "Install" button.

10. Click the "Finish" button on the successful Installation window. (Do not add an account, this is not necessary).

11. Citrix Workspace App is now installed on your computer and ready to use.

This document outlines installing the Citrix Workspace App for a Mac computer. The Citrix Workspace app runs the Virtual Applications & Desktops and must be installed on your computer for them to work.

1. Open your internet browser of choice (we suggest Google Chrome).

2. Navigate to https://www.citrix.com/downloads/workspace-app/mac/workspace-app-for-mac-latest.html.

3. Click the "Download Citrix Workspace app for Mac" button to download the installer file.

4. Save the file to your downloads folder (Safari does this by default).

5. Navigate to your "Downloads" folder and double-click the CitrixWorkspaceApp.dmg file.

6. Double-click the "Install Citrix Workspace" button.

7. Click the "Continue" button on the "This package will run a program to determine if the software can be installed" prompt.

8. Click the "Continue" button on the Welcome to the Citrix Workspace Installer window.

9. Click the "Continue" button on the Welcome to the Software License Agreement window.

10. Click the "Agree" button to agree to the terms of the software license agreement.

11. Click the "Install" button to install to the default location.

12. Type in your Mac Username and Password (not Apple ID username/password) and then click the "Install Software" button.

13. Leave the "Add Account" check box unchecked and click the "Continue" button.

14. Click the "Close" button.

15. Click the "Move to Trash" button (this will delete the download files to save hard drive space).

16. Citrix Workspace App is now installed on your Mac and is ready to be used.

Note that Citrix apps run on a Windows server, like running a virtual Windows machine. Therefore, we recommend using Windows keyboard shortcuts (e.g., Ctrl+C instead of Cmd+C to copy selected texts) while using the apps or modifying your shortcut settings for the apps.

Because the Citrix system is like accessing a remote computer, you cannot use the app until you re-establish your connection. However, the remote session will continue for up to 1 hour. The next time you connect within that hour, your session will open in the same state as when you lost your connection.

For Mac users::

1. Open the Citrix Workspace App.

2. In the upper left corner of your screen, click on Citrix Workspace and select Preferences next to the Apple symbol.

3. In the Preferences window, click on the Accounts tab, then select the Primary store account shown in the middle of the window. Next, click the Minus button and OK to confirm removing the account.

4. Close the Preferences window and click Set Up Account. In the Add account window, enter the following https://myuntlab.cloud.com.


For Windows users:

1. Launch the Citrix Workspace App from your Start Menu.

2. After the Citrix Workspace has launched, click on the Workspace icon, and select Advanced Preferences.

3. Click on Reset Citrix Workspace.

4. You will get a pop-up asking if you want to reset the Citrix Workspace and click yes.

5. Select OK, and your Citrix Workspace should automatically relaunch.
