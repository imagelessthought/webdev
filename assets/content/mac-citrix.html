Using Citrix Workspace on a Mac Computer

MyUNTLab: Citrix Workspace

The Citrix Workspace app for the MacOS does not work reliably at this time. We have removed it from the lab and classroom computers.

But you may still use Citrix Workspace on a Mac: you will just need to always use a browser to access it.

Remember, Citrix Workspace is available from any device, on or off campus. All you need to use Citrix (or Guacamole) is a device that is capable of running a HTML5-capable web browser.

How to Access Citrix Workspace through a Browser

Below are instructions for logging into Citrix Workspace using a browser.

Begin by going to the UNT MyLab website.


Click on the application you want to use and it will launch in the browser. When launching an app for the first time it may take a little while to initialize - please be patient.
