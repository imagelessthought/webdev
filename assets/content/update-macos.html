Updating your MacOS Device

This tutorial provides step-by-step instructions for installing updates on your Mac device.

We will not be sending Nudge notifications until next week but you are free to install it now if you like, and you won't see those notices at all. Nudge Notifications for Mandatory Updates

When mandatory updates become available for your MacOS device, we use a notification system known as Nudge to alert our users that they are required to install these updates. These are official notifications and the updates need to be installed as soon as you are able to do so. A period of time will be specified in the notice indicating how long you will have to install the updates, after which the updates will be installed automatically. The device will be unavailable for you to use during the update so to avoid any inconveniences, please apply the updates when most convenient to you and before the time has expired. An email will be sent to your official UNT email address notifying you that a required update is available and that you will be receiving a notification."

Depending on the severity of the issue that the update addresses, the time before the update is automatically installed could be between 2 days and 2 weeks.

You can choose to install the update directly from the notification by clicking the Update Device button in the top right of the notification.

You can choose to defer the update and install it later, at a more convenient time by clicking the defer button at the bottom right. If you choose to defer, you will need to follow the instructions for Self Service, which are provided in the next section.

The device must be plugged in to an external power source and be connected to the internet for the update to be installed.

If the device is not able to install the update (not connected to power, not connected to the internet or there is not enough drive space for the update) your device will not allow you to use it until it is physically brought into our office, it will not be possible to install the update remotely.

  • Be aware, upgrading the computer can take anywhere from 30 minutes to 2 hours depending on network speed and/or age of computer. Please be prepared to allot adequate time to the process.
  • Be sure to save all open files and close all applications prior to starting the update process.

Note: When the computer starts back up, it may present a series of available options in the new operating system (such as Accessibility options, Data & Privacy, TouchID, and Light/Dark mode).

How to Launch the Upgrade in Self Service

If you choose to defer the update, or are installing an available update prior to the Nudge notifications, these are the steps to take to begin the process when you are ready to do so.

From the Applications menu, launch Self Service.

Select Operating Systems from navigation pane on the left and click the Update button.

Post Upgrade Steps in System Preferences

Apple has released an update, Rapid Security Responses, that was unfortunately not included in the MacOS Ventura Upgrade. Fortunately, is a very quick installation. You will see an announcement like the one below after you install the MacOS Ventura Upgrade. You are able to install it by clicking on the announcement by clicking on it the announcement and following the instructions. If you dismiss the announcement or need to confirm that it is installed, simply open up System Preferences > General > Software Update and Check for Updates if you do not see it available as an option. Run the Check for Updates just to be certain.

You will be prompted for a login, but this is not a request for administrator credentials, you can use your own EUID and password to install the update.
