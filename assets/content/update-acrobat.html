Adding, Removing, and Updating Adobe Creative Cloud Apps

Adobe Acrobat

All CVAD computers come with Acrobat Reader preinstalled. If the full Acrobat Pro application is not installed on your computer and it is required (or if it has been removed and needs to be reinstalled), it can be installed through the Adobe Creative Cloud application. The steps below will walk you through installing the app.

If you have an out-of-date version of the Acrobat application with a severe security vulnerability installed on your device, it will be automatically removed from your device to mitigate the issue. When you follow the steps below, the latest version of the app will be installed.

The instructions below can be used to add, remove, or update any Adobe Creative Cloud appplication that is included in the UNT Adobe agreement. The screen prints are from MacOS, but the steps include instructions for Windows devices as well.

Locate the Creative Cloud application in the applications folder. If it is not on your desktop, you can find it under Applications on an Apple device, or on the Start Menu on a Windows Device.

This is the icon for the Adobe Creative Cloud.

All of the installed applications appear at the top, along with a message indicating whether it is up-to-date. If it is not up-to-date, you will have the option to Update the application.

By clicking on the three dots to the right of the Open button, you can Uninstall the application or Manage plugins.

To install an application, locate the application within the Creative Cloud application under Available in your plan and click Install. By clicking on the three dots immediately to the right you can choose to install previous versions off the app: this is rarely needed but you should be aware of the option in case you encounter a problem in the currently installed version.
