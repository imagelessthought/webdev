Tutorial: Using Citrix for Remote Access

Remote connections to Windows computers, or streamed applications, on the UNT campus are provided by Citrix Workspace. Citrix Workspace is a universal software client that provides secure, high-performance delivery of virtual desktops and applications. Clients are available for all major operating systems and accessible using any modern web browser that supports HTML5.

The Citrix Workspace app for the MacOS does not work reliably at this time. We have removed it from the lab and classroom computers. A tutorial for accessing Citrix from a web browser on a Mac can be found here.

Getting Started

Below you will find instructions to help get you started on Citrix Workspace.

Visit Multi-Factor Authentication (DUO) for help on how to get started.

To quickly access the Windows desktop and applications from MyLab using your web browser, simply visit http://myuntlab.cloud.com and log in with your UNT email address and password. Select Use web browser when you reach the UNT Citrix Workspace welcome page.

To use the full set of features bundled with Citrix Workspace, you will want to use the desktop client. Simply visit http://myuntlab.cloud.com and log in with your UNT email address and password. Select Detect Workspace when you reach the UNT Citrix Workspace welcome page. The website will attempt to detect an existing Citrix Workspace installation and launch the application. If not yet installed, please review the license agreement, check the I agree box and select Download to install the Citrix Workspace desktop client.

Web browser: you may upload files from, and download files to, your local computer in two ways:

1:  Use the Citrix menu at the top-center of your session to select the Upload and Download buttons

2:  Drag-and-drop files from your local computer onto your web browser-based application/desktop to trigger the Upload to prompt where you may select where you want the file(s) uploaded and utilitize the Save To My Device option when saving files which will trigger your local browser to download the file.



3:  Additionally, if using the UNT Desktop you will find shortcuts on the desktop to your UNT OneDrive cloud storage as well as browsers you may use to access your personal, 3rd party storage solution. Citrix desktop client: using the Citrix desktop client simplifies file access by making your local computer's storage devices and network shares accessible within the operating system's and the applications' native File Explorers.
